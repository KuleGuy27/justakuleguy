Everytime a new YT short is made:
- Make a folder with the same name as the video project files
- copy the DTDV (Date, title, description, video) of vid.txt
- copy the thumbnail.jpg (or png)
- copy the .srt subtitle files (if have)
- make a video.html file

 - in the youtube shorts list (index), copy the previous block and change the shorts #, add the a 'a href' link, add the thumbnail source, duration time, change the title and date

- in the HTMl short videos itself; edit the site title from the folder, video source, h1 (vid title), h3 date, h3 description, link url with "a href" if possible
- add downloadable links to download the .srt subtitles + add a mirror link (if have)
- link to mirrors (YouTube, Internet Archive)
- paste video sources
