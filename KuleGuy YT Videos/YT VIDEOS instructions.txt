Everytime a new YT video is made:
- Make a folder with the same name as the video project files
- copy the DTDV (Date, title, description, video) of vid.txt
- copy the thumbnail.jpg (or png)
- copy the .srt subtitle files (if have)
- make a video.html file

 - in the youtube videos list (index), add the a 'a href' link, add the thumbnail source, change the title and date

- in the HTMl videos itself; edit the site title from the folder, h1 (vid title), h3 date, video source, h3 description, link url with "a href" if possible
- add downloadable links to donwload the .srt subtitles + add a mirror link
- link to mirrors (YouTube, Odysee, Rumble, Internet Archive)
