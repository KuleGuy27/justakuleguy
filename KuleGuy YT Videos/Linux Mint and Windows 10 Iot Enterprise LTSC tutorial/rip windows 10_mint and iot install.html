Windows 10 is dead... but there are options!

27 Oct 2025

On the 14th of October 2025. Microsoft is ending their support for Windows 10 and this is a huge deal since a lot of people use Windows 10 and now tons of computers are vunerable because people will stop receiving security updates. Because of this, I wanted to make a SomeOrdinaryGamers style video to show you guys other operating systems that you can install because Windows 10 is now dead. Get Linux Mint here: https://www.linuxmint.com/
Rufus download: https://rufus.ie/en/
Install Wine for Linux: https://gitlab.winehq.org/wine/wine/-/wikis/Download

Get Windows 10 Iot Enterprise LTSC 2021: https://massgrave.dev/windows_ltsc_links
-Install WoeUSB on Linux Mint:
sudo add-apt-repository ppa:tomtomtom/woeusb
sudo apt update
sudo apt install woeusb woeusb-frontend-wxgtk
-IF WOEUSB DOES NOT WORK:
sudo fdisk -l
sudo umount [/yourdrive]
sudo mkfs.ntfs [/yourdrive]
